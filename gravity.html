Gravity Simulator

Choose the properties you want a gravitational body to have using the options, then click on the canvas to create a body with those properties. Once you have added all the bodies you want, click the start button to begin the simulation.

The velocity sliders start in the center with 0 velocity. The slider pointer will turn green to indicate that a body has a starting velocity, and will turn grey if there is no velocity.
Ticking the "lines" checkbox enables trails behind each body, which will be the same colour as the body.
